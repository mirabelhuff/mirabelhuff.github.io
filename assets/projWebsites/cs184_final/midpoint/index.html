Project Overview

As a reminder, for our final project, we wanted to implement physically based simulations of fluids in real time. Our plan was to first implement a water simulator (based on this paper by Miles Macklin and Matthias Muller ) and then add on different kinds of fluids, such as mercury or chocolate.

You can find our project proposal here.

Project Status

Our goal for the first week was to have a simulation environment that we woud be able to visualize and test our work in. We got some starter code to help with visualizing the water particles from the instructors, which was great, but it took us some time to get it working on our computers, especially because Mirabel is working in a Linux environment while Anna is working in a Mac environment. For both of us, there were a lot of things we needed to install, such as GLM and OpenCV and we ran into installation issues with both packages. We ended up making changes to the MakeFile we were given in order to properly run openCV and Anna had to change hers more in order to deal with some deprecated functions that were preventing her from compiling completely. In the end, once we got things to compile, we still had slightly different results on our different systems even though we were running the exact same things. Since it was just a color issue, we decided to continue.

We also have the set up to automatically create videos of our simulation as it happens. However, since we currently don't have any simulation, nothing really gets recorded except for camera changes.

What We Plan To Do Next

Originally, by this point, we had planned to have the water simulator complete (minus collisions) so that we could start working on different types of fluids and possibly even some shaders and renders. Given our current status, we're changing what we plan to deliver a little bit as we think we overestimated how much we could complete with this project. Below you can find our updated schedule.

Updated Schedule

Now (2 Weeks In)

So far, we've set up a simulation environment to test and visualize our simulation.

Week 3

This week, we plan to actually implement the simulation. We've planned out what we need to do pretty thoroughly so we don't expect it to take that long to actually implement our plan. We're planning to use a HashMap like we did for the cloth simulation project to determine the particles in relation to each other. Then, we wil luse the paper linked above to apply collisions to the particles and update the hash map at each step.

Week 4

In the final week, we plan to add different kinds of fluids aside from water by changing the properties of the forces acting on the particles.
